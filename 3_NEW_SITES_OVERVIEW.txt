DEPLOY TO RENDER (45 minutes total):
───────────────────────────────────

  Step 1: Push to GitHub
  ├─ git add .
  ├─ git commit -m "Deploy Fortis, Nexus, Cipher"
  └─ git push origin main

  Step 2: Create Render Services (3x)
  ├─ https://dashboard.render.com
  ├─ New Web Service → Connect GitHub
  ├─ Build: pip install -r {site}/requirements.txt
  ├─ Start: cd {site} && python app.py
  ├─ Env: SECRET_KEY, DEBUG=False
  └─ Deploy!

  Step 3: Test Live Sites
  ├─ https://fortis-banking.onrender.com
  ├─ https://nexus-capital.onrender.com
  └─ https://cipher-wealth.onrender.com
