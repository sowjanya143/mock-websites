## Site Characteristics & Anti-Scraping Features

### 1. Sentinel Capital Partners
- **Focus:** Alternative assets (real estate, credit, infrastructure)
- **Cookie Banner:** Mandatory (data blocked until acceptance)
- **Anti-Scraping:** CAPTCHA on every page, requires JS, requires cookies
- **Special Features:** Multiple CAPTCHA challenges on data pages

### 2. Apex Investment Group
- **Focus:** Growth equity and venture capital
- **Cookie Banner:** Mandatory (data blocked until acceptance)
- **Anti-Scraping:** Rate limiting (6 req/60s), requires JS, requires cookies
- **Special Features:** API endpoint for AUM data

### 3. Meridian Global Holdings
- **Focus:** Multi-strategy institutional investing
- **Cookie Banner:** Optional (data accessible, banner dismissible)
- **Anti-Scraping:** Random CAPTCHA (20% chance), artificial delay (0.5s), requires JS
- **Special Features:** Popup notifications on data pages

### 4. Premier Financial Services
- **Focus:** Value equity investing
- **Cookie Banner:** Optional (data accessible, banner dismissible)
- **Anti-Scraping:** Requires JS, requires cookies (set-cookie middleware)
- **Special Features:** Clean, fast-loading site

### 5. Zenith Asset Management
- **Focus:** Multi-strategy alternatives
- **Cookie Banner:** Optional (data accessible, banner dismissible)
- **Anti-Scraping:** CAPTCHA on data pages, rate limiting, requires JS, artificial delay (0.5s)
- **Special Features:** Complex scraping challenge, API endpoints, popups

### 6. Fortis Banking Group
- **Focus:** Credit investments
- **Cookie Banner:** None (CAPTCHA-based instead)
- **Anti-Scraping:** CAPTCHA on selected pages, requires JS, set-cookie middleware
- **Special Features:** CAPTCHA required for strategies, investor resources, funds pages

### 7. Nexus Capital
- **Focus:** Systematic/quantitative strategies
- **Cookie Banner:** None (CAPTCHA-based instead)
- **Anti-Scraping:** Bot detection, honeypot fields, strict user-agent checks, requires JS
- **Special Features:** Advanced bot detection with honeypot traps

### 8. Quantum Funds
- **Focus:** Derivatives and absolute return
- **Cookie Banner:** None (CAPTCHA-based instead)
- **Anti-Scraping:** IP/geographic blocking (mock), requires JS
- **Special Features:** GeoIP checks, VPN detection, rotating IP blocks (mock)

### 9. Cipher Wealth Management
- **Focus:** Global macro investing
- **Cookie Banner:** None (CAPTCHA-based instead)
- **Anti-Scraping:** CAPTCHA on every page, content obfuscation, requires JS
- **Special Features:** Distorted CAPTCHAs (5 chars), aggressive anti-scraping

## Testing Endpoints

### Navigation Testing
- Each site has 8 main routes: `/`, `/about`, `/leadership`, `/strategies`, `/investor-resources`, `/funds`, `/news`, `/contact`
- Pagination on `/leadership` (5 members per page)
- Individual fund detail pages at `/fund/<id>` where id is 1-5

### Cookie Banner Testing
- **Mandatory sites** (Sentinel, Apex): Data blocked until cookies accepted
- **Optional sites** (Meridian, Premier, Zenith): Banner shown but data accessible
- **No banner sites** (Fortis, Nexus, Quantum, Cipher): Traditional CAPTCHA obstacles

### Anti-Scraping Testing
- Visit any site with: `curl`, `wget`, `requests` library
- Expected: 403 Forbidden or CAPTCHA challenge
- Visit with proper browser headers: Should work normally

### CAPTCHA Testing
- **Mandatory CAPTCHA sites**: Visit `/strategies` or `/funds` - CAPTCHA appears
- **Random CAPTCHA sites**: Refresh page multiple times - sometimes CAPTCHA appears
- **Every page CAPTCHA sites**: Any page may show CAPTCHA

### API Testing
- Apex: `https://apex-mock-website.onrender.com/api/aum` (funds data)
- Zenith: `https://zenith-mock-website.onrender.com/api/funds` (funds list)
- Other sites: `/aum` endpoint returns 403 (blocked)

## Deployment Status

All 9 sites follow the pattern:
```
https://{site-name}-mock-website.onrender.com/
```

Where `{site-name}` is one of:
- sentinel
- apex
- meridian
- premier
- zenith
- fortis
- nexus
- quantum
- cipher

To deploy to Render, each site directory contains:
- `Procfile` - Gunicorn configuration
- `runtime.txt` - Python version specification
- `requirements.txt` - Python dependencies
- `app.py` - Flask application entry point
- `config.py` - Site configuration
- `data/` - JSON data files
- `templates/` - Jinja2 templates
- `static/` - CSS and JavaScript files
